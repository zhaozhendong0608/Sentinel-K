# 💾 05_归档重启 (Archiving & Reset V2.0)

> **定位**: 任务周期的终点。
> **目的**: 固化知识，防止 LLM 上下文漂移，强制重启。
> **V2升级**: 集成 `Asset_Update` (ADR/Landscape) 与 `Pack-0 Reset`。

## Node A: 资产更新 (Asset Update)

**动作**:
1.  **Sync ADR**: 将本次会话产生的 ADR Draft 转正为 `Active`，更新 `Sentinel-K_内核/03_决策日志 (ADR).md` 索引。
2.  **Sync Landscape**: 若架构有变，更新 `Sentinel-K_内核/01_项目全景 (Landscape).md`。
3.  **Sync Task**: 更新 `Sentinel-K_内核/05_项目进度 (Tasks).md` 的状态。
4.  **Sync FAQ**: 识别本次会话中的核心技术见解及 Q&A，沉淀至 `Sentinel-K_内核/04_答疑库 (FAQ).md`。

## Node B: 战报结算 (Battle Report)

**输出 `Archiving Output`**:
```markdown
### 💾 Archiving Output (Reset)
- **Session_Summary**: [本次交付价值]
- **Key_Decisions**: [新增/更新 ADR]
- **Evidence_Ref**: [Run/Scan 报告链接]
- **Asset_Update**: [LANDSCAPE/SPEC 的状态]
```

## Node C: 物理重启 (Physical Reset)

向用户发出 **最终指令**:

> "🛑 **SYSTEM RESET**
>
> *   资产已同步。
> *   上下文 Pack 已卸载。
> *   **建议操作**: 请开启新会话 (New Chat) 并从 `01_启动自检` (Pack-0) 重新开始。"
